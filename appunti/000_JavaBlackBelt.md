# Griglia di Classificazione Competenze Java Junior Developer

*(Scala ispirata alle cinture delle arti marziali - corrispondenza indicativa con la valutazione da 18 a 30)*

L'obiettivo è fornire una progressione graduale che consenta di valutare la crescita tecnica di uno sviluppatore Java, dalla comprensione delle basi dell'OOP fino alla realizzazione di applicazioni enterprise con Spring Boot.

| Livello    | Voto  | Descrizione                                    |
| ---------- | ----- | ---------------------------------------------- |
| ⚪ Bianco   | 18-20 | Conoscenze fondamentali                        |
| 🟡 Giallo  | 21-22 | Programmazione procedurale e OOP base          |
| 🟠 Arancio | 23-24 | Modellazione a oggetti                         |
| 🟢 Verde   | 25-26 | Programmazione orientata agli oggetti avanzata |
| 🔵 Blu     | 27    | Sviluppo applicativo completo                  |
| 🟤 Marrone | 28-29 | Sviluppatore junior avanzato                   |
| ⚫ Nero     | 30    | Java Developer autonomo                        |

---

## Conoscenze

* Operatori
* Input/output da console
* Costrutti condizionali
* Cicli

### Classi

* Creazione di una classe
* Attributi
* Metodi
* Costruttore base

### Incapsulamento

* Attributi privati
* Getter e Setter

### Esempi che sa realizzare

* Calcolatrice
* Gestione studente
* Rubrica semplice

### Autonomia

Lavora solo su esercizi guidati.
