# 2. Stream API

---

# Cos’è uno Stream

Pipeline per elaborare dati:

* filter;
* map;
* reduce;
* collect.

---

# Filter

```java id="4v5mjp"
numeri.stream()
       .filter(n -> n > 10)
       .forEach(System.out::println);
```

---

# Map

```java id="c7v8mo"
nomi.stream()
    .map(String::toUpperCase)
    .forEach(System.out::println);
```

---

# Collect

```java id="9h1a3x"
List<String> risultato =
    nomi.stream()
        .filter(n -> n.length() > 4)
        .toList();
```

---

# Reduce

```java id="n5q1gl"
int somma =
    numeri.stream()
           .reduce(0, Integer::sum);
```

---

# Pipeline completa

```java id="przkgm"
List<Integer> risultati =
    numeri.stream()
           .filter(n -> n % 2 == 0)
           .map(n -> n * 2)
           .sorted()
           .toList();
```

---

# Esempio reale

## Gestione studenti

```java id="67v4f8"
studenti.stream()
    .filter(s -> s.media() >= 18)
    .sorted(
        Comparator.comparing(
            Studente::cognome
        )
    )
    .forEach(System.out::println);
```

# INSERT con PreparedStatement

```java id="0h4p7x"

    conn.prepareStatement(
        "INSERT INTO studenti VALUES (?, ?, ?)"
    );

ps.setString(2, "Mario");
ps.setDouble(3, 29);

ps.executeUpdate();
```

# Perché PreparedStatement?

* sicurezza;
* prevenzione SQL Injection;
* performance.
