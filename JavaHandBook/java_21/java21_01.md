# 4. Strutture Condizionali

---

# If

```java
if (eta >= 18) {

    System.out.println("Maggiorenne");

}
```

---

# If / Else

```java
if (voto >= 60) {

    System.out.println("Promosso");

} else {

    System.out.println("Bocciato");

}
```

---

# Switch moderno

```java
var giorno = 3;

var nome = switch (giorno) {

    case 1 -> "Lunedi";
    case 2 -> "Martedi";
    case 3 -> "Mercoledi";
    default -> "Altro";

};

System.out.println(nome);
```

---

# Pattern Matching base

```java
Object obj = "ciao";

if(obj instanceof String s) {

    System.out.println(s.toUpperCase());

}
```

# 5. Cicli

---

# While

```java
int i = 0;

while(i < 5) {

    System.out.println(i);
    i++;

}
```

---

# For

```java
for(int i = 0; i < 10; i++) {

    System.out.println(i);

}
```

---

# Foreach

```java
int[] numeri = {1,2,3,4};

for(int n : numeri) {

    System.out.println(n);

}
```
