# 3. Costruttori

---

# Costruttore base

```java id="wrlv2v"
public class Studente {

    String nome;

    Studente(String nome) {

        this.nome = nome;

    }

}
```

---

# Uso

```java id="6qswq0"
Studente s = new Studente("Marco");
```

---

# `this`

```java id="v9x4yn"
this.nome = nome;
```

Serve a distinguere:

* attributo;
* parametro.

---

# Costruttori multipli

```java id="sy8v23"
Studente() {}

Studente(String nome) {}
```

# 4. Incapsulamento

---

# Problema attributi pubblici

```java id="0vuzvf"
s.eta = -100;
```

---

# Soluzione: private

```java id="mvxj1m"
private int eta;
```

---

# Getter e Setter

```java id="2n55ga"
public int getEta() {

    return eta;

}

public void setEta(int eta) {

    if(eta >= 0) {

        this.eta = eta;

    }

}
```

---

# Classe completa

```java id="zkvyn7"
public class Prodotto {

    private String nome;
    private double prezzo;

    public Prodotto(String nome, double prezzo) {

        this.nome = nome;
        this.prezzo = prezzo;

    }

    public double getPrezzo() {

        return prezzo;

    }

}
```

# 5. Ereditarietà

---

# Classe base

```java id="7az24w"
public class Animale {

    void verso() {

        System.out.println("Verso generico");

    }

}
```

---

# Classe derivata

```java id="c6t8ye"
public class Cane extends Animale {

    void abbaia() {

        System.out.println("Woof");

    }

}
```

---

# Uso

```java id="4xhy69"
Cane c = new Cane();

c.verso();
c.abbaia();
```

---

# Override

```java id="f6i8n7"
@Override
void verso() {

    System.out.println("Bau");

}
```

---

# `super`

```java id="p2z26w"
super.verso();
```

# 7. Record (Java moderno)

---

# Record semplice

```java id="0xv50q"
public record Persona(
    String nome,
    int eta
) {}
```

---

# Uso

```java id="ctjj2u"
Persona p = new Persona("Anna", 30);

System.out.println(p.nome());
```

---

# Vantaggi

Genera automaticamente:

* costruttore;
* getter;
* `equals`;
* `hashCode`;
* `toString`.

---

# Record vs Classe classica

| Classe         | Record     |
| -------------- | ---------- |
| Verbosa        | Compatta   |
| Mutabile       | Immutabile |
| Getter manuali | Automatici |
